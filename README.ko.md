# CommandBar

**바이브 코딩 시대의 필수 도구**

AI와 함께 개발하는 시대, 명령어와 코드 스니펫을 끊임없이 복사하게 됩니다. CommandBar는 모든 것을 캡처하고, 즉시 재사용 가능하게 만들어 워크플로우를 유지합니다.

[English](README.md)

---

## 왜 CommandBar인가?

**문제**: AI가 명령어와 코드를 생성하는 속도가 정리하는 속도보다 빠릅니다. 같은 curl 명령을 반복하고, 같은 git 워크플로우를 되풀이하고, 무엇이 작동했는지 잊어버립니다.

**해결**: CommandBar가 혼돈을 시스템으로 바꿉니다:
- **한 번 저장, 영원히 실행** - 모든 명령이 재사용 가능한 버튼이 됩니다
- **스마트 클립보드** - 복사하는 모든 것을 자동 캡처 (최대 10,000개)
- **실행 히스토리** - 언제 무엇을 실행했는지 기록
- **파라미터** - 변수를 추가해 유연하게 사용
- **항상 접근 가능** - 화면 위에 떠있어 필요할 때 바로 사용

## 설치

### Homebrew (권장)
```bash
brew tap yejune/command-bar
brew install --cask command-bar
```

### DMG 다운로드
1. [Releases](https://github.com/yejune/command-bar/releases)에서 DMG 다운로드
2. DMG 열고 CommandBar를 Applications로 드래그
3. 첫 실행: 우클릭 → 열기 (Gatekeeper 우회)

### 소스에서 빌드
```bash
git clone https://github.com/yejune/command-bar.git
cd command-bar
make build
```

## 빠른 시작

### 1. 첫 번째 명령 추가하기

`+` 클릭 → 제목 입력 → 실행 타입 선택 → 저장

**예시**: 자주 쓰는 curl 명령 저장
```
제목: 서버 상태 확인
타입: 스크립트
명령: curl -s https://api.example.com/health
```

### 2. 명령 실행하기

- **더블클릭**으로 바로 실행
- **우클릭** → 실행으로 옵션 확인
- **Enter 키**로 선택된 항목 실행

### 3. 파라미터 사용하기

`{파라미터명}` 문법으로 명령을 유연하게:

```bash
# 텍스트 입력
echo "Hello {name}"

# 드롭다운 선택
git checkout {branch:main|develop|feature}

# 혼합 파라미터
curl -X {method:GET|POST} https://api.example.com/{endpoint}
```

명령 실행 시 각 파라미터 입력을 요청합니다.

## 실행 타입

| 타입 | 용도 | 특징 |
|------|------|------|
| **터미널** | 인터랙티브 명령 | iTerm2/Terminal에서 열림, 출력 히스토리 저장 |
| **스크립트** | 빠른 스크립트 실행 | 파라미터 프롬프트, 리사이즈 가능한 결과 창 |
| **백그라운드** | 모니터링 & 자동화 | 조용히 실행, 자동 반복 (10분~7일) |
| **일정** | 알림 & 리마인더 | 날짜/시간 선택, 반복 옵션, 미리 알림 |
| **API** | REST API 테스트 | 모든 HTTP 메서드, 헤더, 바디, 파일 업로드 |

## 클립보드 관리자

CommandBar가 복사하는 모든 것을 자동 캡처합니다:

- **자동 캡처**: 클립보드 실시간 모니터링
- **검색**: 복사한 텍스트 즉시 찾기
- **즐겨찾기**: 중요한 클립 별표로 빠른 접근
- **달력 필터**: 날짜별 탐색
- **메모로 보내기**: Apple 메모 앱으로 내보내기

**팁**: 바이브 코딩에서는 AI가 생성한 코드를 끊임없이 복사합니다. CommandBar가 모든 것을 검색 가능하게 정리합니다.

## API 테스트

엔드포인트 테스트를 위한 내장 REST 클라이언트:

```
메서드: POST
URL: https://api.example.com/users/{userId}
헤더:
  Authorization: Bearer {token}
  Content-Type: application/json
바디:
  {"name": "{userName}", "role": "admin"}
```

**기능**:
- GET, POST, PUT, DELETE, PATCH 메서드
- 커스텀 헤더 및 쿼리 파라미터
- JSON, Form Data, Multipart 바디 타입
- 파일 업로드 (MIME 자동 감지)
- 응답 히스토리 저장

## 데이터 저장

모든 데이터는 SQLite에 로컬 저장:
```
~/.command_bar/command_bar.db
```

**백업**: 이 파일만 복사하면 모든 명령, 히스토리, 설정이 백업됩니다.

## 바이브 코딩 팁

1. **AI가 준 명령은 바로 저장** - Claude/GPT가 유용한 명령을 주면 잊기 전에 저장하세요
2. **파라미터로 유연하게** - 비슷한 명령 10개 대신, `{변수}`가 있는 하나로 대체
3. **클립보드 히스토리 확인** - 2시간 전에 복사한 코드? 아직 있습니다
4. **프로젝트별 그룹** - 색상으로 프로젝트별 그룹 만들기
5. **백그라운드 모니터링** - 개발 서버 상태 체크 자동화
